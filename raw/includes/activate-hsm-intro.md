# Include: Activate HSM intro paragraph
---

All data plane commands are disabled until you activate the HSM. You can't create keys or assign roles. Only the designated administrators that you assign during the create command can activate the HSM. To activate the HSM, you must download the [Security Domain](/azure/key-vault/managed-hsm/security-domain).
