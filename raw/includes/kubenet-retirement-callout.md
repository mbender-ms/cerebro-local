> [!IMPORTANT] 
> On **31 March 2028**, kubenet networking for Azure Kubernetes Service (AKS) will be retired.
> 
> To avoid service disruptions, **you'll need to** [upgrade to Azure Container Networking Interface (CNI) overlay](/azure/aks/upgrade-aks-ipam-and-dataplane#upgrade-an-existing-cluster-to-azure-cni-overlay) **before that date**, when workloads running on kubenet for AKS will no longer be supported.
