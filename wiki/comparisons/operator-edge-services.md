# Azure Operator and Edge Services

Services for telecommunications operators and edge deployments.

| Service | Purpose |
|---------|---------|
| **Operator Service Manager** | Lifecycle management for network functions on Azure Operator Nexus |
| **Private 5G Core** | Deploy private 4G/5G mobile networks on Azure Stack Edge |
| **Network Function Manager** | Deploy and manage network functions (NFs) on Azure |
| **Azure Extended Zones** | Small Azure footprint in metro areas for low-latency edge |
| **Internet Peering** | Set up peering with Microsoft's global network |
| **Peering Service** | Enhanced internet routing through Microsoft's network |
