# Azure Application Network

Application-level networking for Kubernetes. Includes Application Gateway for Containers (AGC) — managed L7 load balancer and ingress controller for AKS with TLS termination, URL routing, and traffic splitting. Successor to AGIC (Application Gateway Ingress Controller).
