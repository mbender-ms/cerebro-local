# Azure Kubernetes Application Network (Preview)

Application-layer networking service for AKS. Provides observability, metrics, and logging for application traffic within and between Kubernetes clusters. (source: application-network/*.md — 11 articles)

## Capabilities

- Control plane and data plane log collection
- Application network metrics (latency, throughput, error rates)
- Architecture designed for service-to-service communication visibility
- Integrates with Azure Monitor for dashboards and alerts
