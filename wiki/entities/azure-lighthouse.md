# Azure Lighthouse

Cross-tenant management at scale. Service providers and enterprises manage resources across multiple Azure AD tenants from a single control plane. (source: lighthouse/overview.md — 32 articles)

## Key Capabilities

- **Azure delegated resource management** — manage customer/partner resources without switching tenants
- **Cross-tenant visibility** — view and manage resources across tenants in one view
- **Azure Policy across tenants** — enforce compliance at scale
- **Managed service offers** — publish management services to Azure Marketplace

## Use Cases

- MSPs managing multiple customer tenants
- Enterprise IT managing subsidiary tenants
- Consulting firms deploying and monitoring across client environments
