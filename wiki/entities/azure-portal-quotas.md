# Azure Portal and Quotas

## Azure Portal

Web-based console for managing Azure resources. Provides dashboards, resource browsers, Cloud Shell, and guided experiences. Alternative to CLI, PowerShell, and REST API. (source: azure-portal/*.md — 30 articles)

Key features: customizable dashboards, favorites, resource groups view, cost management integration, accessibility support, mobile app.

## Azure Quotas

Service limits and quotas for Azure subscriptions. Some adjustable (request increase), some fixed. (source: quotas/*.md — 15 articles)

Key concepts:
- **Adjustable quotas** — request increase through portal or support
- **Non-adjustable quotas** — hard limits set by Azure
- **Usage alerts** — notifications when approaching quota limits
- **Regional quotas** — some limits are per-region
