# Azure DevTest Labs

Quickly create development and test environments in Azure with reusable templates, cost controls, and automated shutdown. Designed for dev/test scenarios where self-service VM provisioning with guardrails is needed.

## Key Features

| Feature | Description |
|---------|-------------|
| **Reusable templates** | Custom images and formulas for consistent environments |
| **Cost controls** | Auto-shutdown, VM size limits, per-user quotas |
| **Artifacts** | Install tools/software on VMs at creation (e.g., VS Code, Docker) |
| **Claimable VMs** | Pre-created pool of VMs users can claim on demand |
| **Integration** | Azure Pipelines, GitHub Actions for CI/CD environments |
| **Policies** | Allowed VM sizes, max VMs per user, auto-shutdown schedule |
