# Troubleshooting: Azure Kubernetes Service

188 troubleshooting articles covering the most common AKS issues.

## Issue Categories

| Category | Common Issues |
|----------|--------------|
| **Connectivity** | Pod-to-pod, pod-to-service, egress/SNAT exhaustion, DNS resolution |
| **Node health** | NotReady, allocation failures, disk pressure, OOM |
| **Networking** | CNI errors, NSG misconfig, load balancer issues, ingress failures |
| **Identity/auth** | Managed identity, RBAC, kubelet credential rotation |
| **Upgrades** | Version upgrade failures, PDB conflicts, drain timeouts |
| **Storage** | Persistent volume mount failures, CSI driver issues |
| **Scaling** | Cluster autoscaler stuck, HPA not triggering |
