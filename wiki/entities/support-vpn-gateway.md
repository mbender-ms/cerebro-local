|-------|-------------|
| S2S tunnel down | IPsec/IKE negotiation failure, mismatched parameters |
| P2S connectivity | Client can't connect, certificate issues, DNS resolution |
| Throughput low | Bandwidth below expected for SKU, encryption overhead |
| BGP not working | Peering not established, routes not exchanging |
| Gateway not responding | Gateway in failed state, redeployment needed |
